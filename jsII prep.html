Define a function named addOn2 that takes in an input and increases by 2 if the input is numeric. If the input is Not A Number, then return false.

addOn2("22") // 24

addOn2(null) // false
addOn2(true) // false


            
                function addOn2 (in1) {
                if ( isNaN(parseFloat(in1))) {
                        return false
                } else {
                return parseFloat(in1)+2;
                    }
                }
